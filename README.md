## Design system

- Background: `#15130F` vinyl black, `#221F19` groove
- Accent: `#C81E3A` record-label red
- Text: `#ECE6D8` paper, `#C9C6BC` etching silver
- Type: Big Shoulders (display), Inter (body), JetBrains Mono (data)
- Signature element: 3D disc carousel with real cover art, continuously spinning

## Stack

Pure HTML/CSS/JS. No framework dependencies. Single-file build.
